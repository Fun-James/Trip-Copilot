
# Trip Copilot 快速启动指南

## 一键启动

1. 启动后端：
   ```bash
   cd backend
   python main.py
   ```
2. 启动前端：
   ```bash
   cd frontend
   npm run dev
   ```

## 访问地址
- 前端：http://localhost:3000
- 后端：http://localhost:8000
- API 文档：http://localhost:8000/docs

## 项目结构
```
Trip Copilot/
├── frontend/        # Vue 3 前端
├── backend/         # FastAPI 后端
├── README.md        # 项目说明
```

## 常用命令
- 停止服务：终端窗口按 Ctrl+C
- 重新启动：重复上述启动命令

## 测试方法
1. 打开前端页面
2. 输入目的地，体验智能规划与聊天
