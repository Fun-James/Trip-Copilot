### 🔧 快速操作

#### 停止服务
如果需要停止服务，请在相应的终端窗口按 `Ctrl + C`

#### 重新启动
- **前端**: `cd frontend && npm run dev`
- **后端**: `cd backend && python main.py`

#### 测试功能
1. 打开 http://localhost:3000
2. 在搜索框中输入旅行目的地（如"北京"）
3. 点击发送按钮或按回车键
4. 查看AI助手的回复
